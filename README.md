# WebScan Pro — AI-Powered Website Growth Intelligence

## Quick Start

### Prerequisites
- Node.js 18+
- pnpm (`npm install -g pnpm`)

### Install & Run
```bash
pnpm install
pnpm dev
```
Then open http://localhost:8080

### First-Time Setup
1. Go to **Settings** in the top nav
2. Enter your **Anthropic API key** (get one at https://console.anthropic.com)
3. Click **Save Settings**
4. Start using all AI-powered features!

## Features
- **Scanner** — 6-phase AI website scan with animated progress and score cards
- **AI Agents** — Deploy 6 autonomous agents with live terminal logs
- **Scheduler** — Schedule and automate agent runs
- **Content Hub** — Claude-powered blog generator with 6 templates
- **Keywords** — AI keyword research and gap analysis
- **Analysis** — Deep SEO, AEO, and GEO analysis with grades
- **Auto-Post** — Generate platform-native social posts with live preview
- **Analytics** — Track KPIs and monitor AI engine visibility

## Build for Production
```bash
pnpm build
pnpm start
```
